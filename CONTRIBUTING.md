## كيف تساهم؟

### 1. **الإبلاغ عن الأخطاء (Bugs)**

إذا وجدت خطأ:

1. تأكد من عدم الإبلاغ عنه مسبقاً
2. افتح Issue جديد
3. اكتب عنوان واضح
4. اشرح الخطأ بالتفصيل
5. أرفق صورة أو فيديو إن أمكن

### 2. **اقتراح ميزات (Features)**

إذا لديك فكرة جديدة:

1. افتح Issue جديد بعنوان بداية ب "[REQUEST]:"
2. اشرح الفكرة بوضوح
3. اشرح السبب في فائدتها
4. قدم أمثلة إن أمكن

### 3. **تقديم Pull Request**

#### الخطوات:

```bash
# 1. Fork المستودع
git clone https://github.com/YOUR_USERNAME/ashrabha-app.git
cd ashrabha-app

# 2. أنشئ فرع جديد
git checkout -b feature/your-feature-name

# 3. أضف التعديلات
# قم بتعديل الملفات المطلوبة

# 4. اختبر التطبيق
flutter pub get
flutter run

# 5. Commit التغييرات
git add .
git commit -m "إضافة: وصف الميزة"

# 6. Push الفرع
git push origin feature/your-feature-name

# 7. افتح Pull Request
```

## معايير الكود

### اتبع هذه القواعس:

✅ استخدم **Dart** بشكل صحيح
✅ كتابة **تعليقات واضحة**
✅ **تسميات متغيرات واضحة**
✅ **تنسيق الكود المنتظم**
✅ **اختبر الكود قبل الإرسال**

### مثال لـ Commit Message:

```
إضافة: ميزة جديدة لتذكيرات الماء

هذا التعديل يضيف:
- تنبيهات يومية
- وقت مخصص للتذكير
- إمكانية تعطيل التنبيهات
```
